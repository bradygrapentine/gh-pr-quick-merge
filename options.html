Per-repo default merge method

Highlights the chosen button on each PR row for the matching repo, and auto-selects it when bulk-merging. Stored in chrome.storage.sync so it follows you across browsers.
    Use a Personal Access Token instead
    Needs repo scope (classic) or contents: write + pull_requests: write (fine-grained). Create one at github.com/settings/tokens. Stored locally via chrome.storage.sync.
    ⚠ Token is 30+ days old. Consider rotating it. Paste a fresh token to reset the counter.
    Multi-account note: only one token is supported at a time. To switch accounts, paste a token for the other account here. Multiple simultaneous accounts are not supported.

        Fast mode (list endpoint)

        Skip per-PR detail fetches. Loads faster on large PR lists, but the mergeable-state column is unavailable — single-click merge stays disabled while fast mode is on. Bulk close / label still work.

        Import / export

        Move templates, repo defaults, shortcut overrides, and stale-day preference between browsers. Tokens are never exported.
