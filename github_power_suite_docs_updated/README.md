# GitHub Power Suite (Browser Extension)

A browser extension that enhances GitHub pull request workflows by enabling:
- Inline PR actions (merge, squash, rebase) on /pulls page
- PR scanning, filtering, and prioritization
- Comment navigation and CI visibility
- Batch operations for efficient workflow

Philosophy:
Free layer = Insight + Control
Paid layer (future) = Automation + Workflow orchestration

## Added V1 PR Page Safety Feature

- Always-visible PR page rebase/update button to avoid hidden merge-panel confusion.
