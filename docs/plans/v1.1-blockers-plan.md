## Cross-cutting requirements (apply to every track)

1. Worktrees inside `<project>/worktrees/<slug>` (per global rules), never sibling.
2. Heartbeat contract: `.claude/agent-status/<id>.log` every ~5 min for any agent expected to run >5 min.
3. Local green gate before any `gh pr ready`: `npm test && npm run typecheck && npm run lint`.
4. After parallel dispatch, run `gh pr list --author @me` to confirm shipping; never assume.
5. Each PR ends with the `**Open PRs**` block per global rules.
6. `gh pr merge --auto --squash` exactly once per PR; no retry, no bypass.
